# Test Table - Каталог курсов

Небольшой проект каталога онлайн-курсов с поиском и фильтрацией.

## Что это

Карточный каталог курсов с живым поиском и фильтрацией по категориям. Верстка адаптивная, работает от 320px до 1920px.

## Технологии

- HTML
- JavaScript (vanilla)
- SCSS
- Vite (для сборки)

Без библиотек и фреймворков, только нативный JS.

## Как запустить

1. Установи зависимости:
```bash
npm install
```

2. Запусти dev-сервер:
```bash
npm run dev
```

3. Открой в браузере адрес, который покажет Vite (обычно `http://localhost:5173`)

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`.

## Функционал

- Поиск по названию курсов (живой, без перезагрузки)
- Фильтрация по категориям
- Кнопка "Load more" для подгрузки карточек
- Адаптивная верстка (резиновый адаптив без жестких брейкпоинтов)

## Структура проекта

```
test-table/
├── src/
│   ├── main.js          # Основная логика
│   ├── style/           # SCSS стили
│   │   ├── blocks/      # Блоки по BEM
│   │   └── common/      # Общие стили
│   └── fonts/           # Шрифты
├── public/              # Статические файлы
│   └── svg/             # SVG иконки и декорации
└── dist/                # Собранный проект
```
